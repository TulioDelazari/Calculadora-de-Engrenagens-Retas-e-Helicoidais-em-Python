# Calculadora de Engrenagens

Aplicação desenvolvida em Python para cálculo de parâmetros geométricos de engrenagens retas e helicoidais.

## Funcionalidades

- Relação de transmissão
- Distância entre centros
- Passo circular
- Diâmetro primitivo
- Diâmetro externo
- Diâmetro interno
- Altura dos dentes
- Cálculos para engrenagens helicoidais

## Tecnologias

- Python 3
- Math
- Tkinter

## Objetivo

Este projeto foi desenvolvido com fins acadêmicos para aplicar conceitos de Engenharia Mecânica, Elementos de Máquinas e Transmissão de Potência, automatizando cálculos frequentemente utilizados no dimensionamento de engrenagens.
